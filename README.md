## Form Builder API

This project is an API built for Form Builder application.

## Requirements
- Node.js - minimal v16.0.0
- Mongodb - minimal v4.0.0

## Installation

```bash
$ npm install
```

## Usage (run the server)

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Technologies used
- Nest.js (https://nestjs.com)
- Mongodb (https://www.mongodb.com)
